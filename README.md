# Landing Page - Méthode Trading Mathieu B

Landing page pour la méthode de trading de Mathieu B.

## Description

Site web de landing page avec quiz interactif pour accéder à une méthode de trading gratuite.

## Structure du projet

- `index.html` - Page principale HTML
- `style.css` - Styles CSS
- `script.js` - Logique JavaScript
- `images/` - Images et logos

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)

## Installation

1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur web

## Fonctionnalités

- Écran d'accueil avec présentation
- Quiz interactif
- Formulaire de collecte d'informations
- Vidéo YouTube intégrée
- Modales pour les conditions générales et politique de confidentialité
